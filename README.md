## REQUIREMENTS

Node    v0.10.31  
Bower   v1.3.9  
Grunt   v0.4.5

## INSTALLATION

clone the repo then:  
`npm install`

After all dependencies are installed:  
`grunt serve` to run the task runner  
`grunt build` to build the app into `/dist` for deployment
